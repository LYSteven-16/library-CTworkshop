🔎 Problem Definition: Find the Pain Point

Students cannot find seats → a common pain point worth solving with tech

Problem Def.
✅ Key Takeaway: A good question is the starting point. "Students need real-time crowding info" gave all subsequent work a clear direction.
2

🧅 Decomposition: Break Problems into Modules

Traffic system → Collection module + Processing module + Display module

Decomposition
✅ Key Takeaway:Decomposition lets each module be independently designed and built. Collection uses sensors, processing uses algorithms, display uses interfaces — decoupled, each with its own role.
3

📊 Pattern Recognition: Discover Patterns & Anomalies

Morning peaks, midday dips, Wednesday anomaly → find patterns to predict and decide

Pattern Recognition
✅ Key Takeaway: Data hides three pattern types — periodic (daily peaks/dips), trending (weekends quieter), anomalies (event spikes). Identifying them is the prerequisite for algorithm design.
4

🎨 Abstraction: From Data to Interface

Raw counts → Traffic lights / Heatmap / Suggested times → Selective presentation

Abstraction
✅ Key Takeaway: Abstraction isn't deleting data — it's selective presentation. Traffic lights hide exact counts but speed up decisions; heatmaps preserve spatial info, sacrificing exact numbers.
5

⚙️ Algorithm Design: IR Sensor Counting Logic

IR-A → IR-B = Enter / IR-B → IR-A = Exit + Debounce Cooldown

Algorithm Design
✅ Key Takeaway: Algorithm design is precise step sequences. Sensor blocking order determines direction, debounce prevents false counts — every detail affects accuracy.

🏆 Integration Reflection: Connecting Everything

Integration

Concept Quiz

Test your CT understanding with these 4 questions.

1. Breaking the library traffic system into Collection, Processing, and Display modules is which CT skill?
A. Pattern Recognition
B. Decomposition
C. Abstraction
D. Algorithm Design
Decomposition means breaking big problems into manageable parts. Splitting the system into three independent modules with clear responsibilities — decomposition in action.
2. "Cover IR-A then IR-B = Enter" is which CT skill?
A. Decomposition
B. Pattern Recognition
C. Abstraction
D. Algorithm Design
Algorithm Design is precise step-by-step solutions. "A-then-B = Enter" is a clear conditional rule, combined with debounce to form a complete counting algorithm.
3. Replacing exact numbers with traffic lights to show crowding is which CT skill?
A. Algorithm Design
B. Decomposition
C. Abstraction
D. Pattern Recognition
Abstraction means hiding details and keeping key info. Traffic lights hide exact counts (87 people) but keep the crucial judgment (crowded/ok/free) for instant decisions.
4. "10AM-12PM is peak hour daily" is which CT skill?
A. Abstraction
B. Algorithm Design
C. Decomposition
D. Pattern Recognition
Pattern Recognition means finding recurring patterns in data. Spotting daily morning peaks — that's Pattern Recognition: identifying trends and regularities.
